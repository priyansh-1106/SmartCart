# 🛒 SmartCart Customer Segmentation

## 📌 Project Overview

SmartCart Customer Segmentation is a Machine Learning project that analyzes customer purchasing behavior and groups customers into different clusters based on their spending habits, income, demographics, and engagement patterns.

The project uses data preprocessing, feature engineering, dimensionality reduction, and clustering algorithms to identify meaningful customer segments that can help businesses improve marketing strategies, customer targeting, and personalized recommendations.

---

# 🚀 Features

* Data Cleaning and Preprocessing
* Missing Value Handling
* Feature Engineering
* Outlier Detection and Removal
* Correlation Heatmap Visualization
* One-Hot Encoding for Categorical Features
* Feature Scaling using StandardScaler
* PCA for Dimensionality Reduction
* Elbow Method for Optimal K Selection
* Silhouette Score Evaluation
* Customer Segmentation using:

  * K-Means Clustering
  * Agglomerative Clustering
* 3D Cluster Visualization

# ⚙️ Project Workflow

## 1. Data Loading

The dataset is loaded using Pandas.

## 2. Data Preprocessing

* Missing values are handled.
* Unnecessary columns are removed.
* Datetime columns are converted.

## 3. Feature Engineering

New features are created such as:

* Age
* Customer Tenure
* Total Spending
* Total Children

## 4. Outlier Removal

Extreme values are removed to improve clustering performance.

## 5. Data Encoding & Scaling

* One-Hot Encoding is applied to categorical columns.
* StandardScaler is used for feature normalization.

## 6. PCA Visualization

Principal Component Analysis (PCA) is used to reduce dimensions and visualize data in 3D.

## 7. Optimal Cluster Selection

* Elbow Method
* Silhouette Score

## 8. Clustering

The project applies:

* K-Means Clustering
* Agglomerative Clustering
